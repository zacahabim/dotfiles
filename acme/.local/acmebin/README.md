# Acme scripts

This directory contains a list of utility scripts that can be used with the Acme

  ┌─────────┬────────────────────────────────────────────────────────────────────────┐
  │ Tool    │ Purpose                                                                │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ a+      │ Open files/directories in acme                                         │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ a-      │ Close (delete) the current window                                      │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ acopy   │ Copy selection to system clipboard                                     │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ aexec   │ Run a command, show output in new window                               │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ afmt    │ Auto-format file (detects language: Go, Python, Rust, JS, JSON, C, sh) │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ afinish │ Wait for a window's command to complete                                │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ aget    │ Reload file from disk                                                  │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ agofmt  │ Format Go file with goimports/gofmt                                    │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ agrep   │ Recursive grep with acme-plumbable output                              │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ aindent │ Indent/unindent selection                                              │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ alook   │ Look up a word (dictionary or in-file search)                          │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ aname   │ Get or set window name                                                 │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ apaste  │ Paste clipboard into window                                            │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ aput    │ Save (write) window to disk                                            │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ asel    │ Print current selection to stdout                                      │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ atag    │ Read or append to the window tag                                       │
  ├─────────┼────────────────────────────────────────────────────────────────────────┤
  │ aremote │ Execute program in the remote host if file in a SSHFS mount            │
  └─────────┴────────────────────────────────────────────────────────────────────────┘
